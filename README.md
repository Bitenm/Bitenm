## Projeto em destaque

### [EstoqueApp](https://github.com/Bitenm/EstoqueApp)

Aplicativo Android nativo em Java, criado como projeto de demonstração de arquitetura e boas práticas.

- Autenticação local segura (Room + hash de senha SHA-256 com salt — sem senha em texto puro)
- Consumo de API REST com Retrofit + Gson
- Listagem com busca em tempo real, pull-to-refresh e Material Design
- Arquitetura organizada em camadas (`model`, `network`, `db`, `adapter`)
